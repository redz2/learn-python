# 协程
1. 协程不是计算机提供的，程序员人为创造，也被称为微线程，是一种用户态上下文切换技术。
2. 实现协程的几种方法
    * greenlet，早期模块
    * yield关键字
    * asyncio装饰器（py3.4）
    * async、await关键字（py3.5）目前最主流！！！
3. 协程的意义
    * 在一个线程中如果遇到IO等待时间，线程不会傻傻等，会利用空闲时间再去干点别的
    * [一文搞懂python async/await异步编程](https://zhuanlan.zhihu.com/p/698683843)
4. 协程包含什么
    * 协程本身也是耗时操作
    * IO耗时操作
