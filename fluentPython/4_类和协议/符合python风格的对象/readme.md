## 符合python风格的对象
1. 自定义类型的行为可以和内置类型那样自然
    * 靠的是鸭子类型（dunk typing），而不是继承
    * 实现相应的方法即可
2. 如何实现常见的特殊方法？
    * 对象表示形式
        1. 返回Unicode字符串
            * __repr__: repr()调用，便于开发者理解
            * __str__ : str()调用，便于用户理解
            * __format__: format()和str.format()调用
        2. 返回字节序列
            * __bytes__: bytes()调用，对象的字节序列表示形式
